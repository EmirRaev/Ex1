HashSet нельза сортировть, так как HashSet испльзует HashMap. HashMap использует массив из LinkedList
Внутри LinkedList, состоит из Значения обьекта и кода, HashMap использует обе ячейки для сортировки
HashSet использует туже логику, но записывает Значение в ячейку предназначеную для кода(double), и так как используется
для сохрание и удобного взаимодействия с ним HashCode(int) то результат сортировки может быть не предсказуемым.
